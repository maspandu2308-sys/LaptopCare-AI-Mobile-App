
# LaptopCare AI Mobile App

LaptopCare AI Mobile App is a Vite + React project generated from the original Figma design.

Original design:
https://www.figma.com/design/nVA7ICK6YV9GwHw2dOCByY/LaptopCare-AI-Mobile-App

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- shadcn/ui-style components

## Running Locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
